There is a code that you need to find. You make color guesses and the program will tell you white if a guess is in the right spot, or black if a guess is the right color but in the wrong spot.

Difficult Level:

Easy: The right side pegs will match with the left side. So the if the first peg is white, that means that your first guess is correct. Or if the second peg is black, then that means your second guess is in the wrong spot, but right color.



Medium: This will put the white pegs first and then the black pegs. This means you will NOT know which guess is white/black.



Hard: Same as medium, but with 6 colors.
